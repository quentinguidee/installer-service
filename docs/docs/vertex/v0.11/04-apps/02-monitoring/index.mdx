# Vertex Monitoring

The "Vertex Monitoring" dashboard provides high-level services to monitor the health of Vertex and its applications.

## Getting Started

To configure Vertex Monitoring, you will need to install at least one collector and one visualization app.

This can be done by following these steps:

1. Open the "Vertex Monitoring" app.
2. In the sidebar, click on one of the collectors.
3. Install it by clicking on the "Download" icon, then start it by clicking on the "Play" icon.
4. In the sidebar, click on one of the visualization apps.
5. Install it by clicking on the "Download" icon, then start it by clicking on the "Play" icon.

## Collectors

The following collectors can be used to collect data from Vertex:

- [Prometheus](https://prometheus.io/)

## Visualizations

The following apps can then be used to visualize the metrics:

- [Grafana](https://grafana.com/)
