# Install with binaries

<Tabs>

    <TabItem label="Linux">

        1. Install [Docker](https://docs.docker.com/desktop/install/linux-install/) or [Docker
        Engine](https://docs.docker.com/engine/install/).

        2. Download the precompiled binaries for your system from [GitHub
        Releases](https://github.com/vertex-center/vertex/releases/latest).

        :::info
        - **Raspberry Pi** - Download the `linux_armv6` version.
        :::

        3. Run **Vertex Kernel as sudo**:
        ```Bash
        sudo ./vertex-kernel --user $(whoami)
        ```

        or if you want to use a custom host/port:
        ```Bash
        sudo ./vertex-kernel \
        --host localhost \
        --port 6130 \
        --user $(whoami)
        ```

        4. Open the UI in your browser: [http://localhost:6130](http://localhost:6130)

    </TabItem>

    <TabItem label="macOS">

        1. Install Docker:
        - With [Docker for macOS](https://docs.docker.com/desktop/install/mac-install/)
        - **Or** with Homebrew: `brew install --cask docker`.

        2. Download the precompiled binaries for your system from [GitHub
        Releases](https://github.com/vertex-center/vertex/releases/latest).

        :::info
        - **Intel** - Download the `darwin_amd64` version.
        - **M1/M2** (or newer) - Download the `darwin_arm64` version.
        :::

        3. Run **Vertex Kernel as sudo**:
        ```Bash
        sudo ./vertex-kernel --user $(whoami)
        ```

        or if you want to use a custom host/port:
        ```Bash
        sudo ./vertex-kernel \
        --host localhost \
        --port 6130 \
        --user $(whoami)
        ```

        4. Open the UI in your browser: [http://localhost:6130](http://localhost:6130)

    </TabItem>

    <TabItem label="Windows">

        1. Install [Docker Desktop for Windows](https://docs.docker.com/desktop/install/windows-install/).

        2. Download the precompiled binaries for your system from [GitHub
        Releases](https://github.com/vertex-center/vertex/releases/latest).

        3. Run **Vertex Kernel as administrator**:
        ```Bash
        .\vertex-kernel.exe --user YOUR_USERNAME
        ```

        or if you want to use a custom host/port:
        ```Bash
        .\vertex-kernel.exe \
        --host localhost \
        --port 6130 \
        --user YOUR_USERNAME
        ```

        4. Open the UI in your browser: [http://localhost:6130](http://localhost:6130).

    </TabItem>

</Tabs>

:::tip About sudo
To learn more about permissions, read [Security > Vertex Kernel](../security/vertex-kernel).
:::
