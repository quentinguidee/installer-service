# Report a bug

To report a bug, please create a new issue on one of our repository, including:

|                                                                     | Case                                                                                             |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [Vertex](https://github.com/vertex-center/vertex)                   | An issue related to the functionality of Vertex.<br/>_eg. Deleting a container throws an error._ |
| [Vertex WebUI](https://github.com/vertex-center/vertex-webui)       | An issue related to the web interface<br/>_eg. A button is not clickable._                       |
| [Vertex Services](https://github.com/vertex-center/vertex-services) | An issue related to a service.<br/>_eg. Port binding not working in the Postgres service._       |
| [Vertex Documentation](https://github.com/vertex-center/docs)       | An issue related to the documentation.<br/>_eg. A typo in the documentation._                    |
