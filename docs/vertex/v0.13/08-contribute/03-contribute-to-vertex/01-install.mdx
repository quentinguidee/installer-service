# Install Vertex to develop locally

- To install Vertex for development, please follow the instructions to install from sources: [Install > Install from sources](./../../install/install-from-sources).
- You might also need to [install the Vertex WebUI](./../contribute-to-vertex-webui/install) to interact with the latest version of the Vertex API.

Everything is set up for you to start developing!
