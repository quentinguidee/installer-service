# Install for development

To install Vertex for development, you will need to install both the Vertex Client and the Vertex Server.

---

## Install the Server

1. Install Go using the [official documentation](https://go.dev/doc/install).

2. Download the project
    ```bash
    git clone https://github.com/vertex-center/vertex.git
    cd vertex
    ```

3. Run **Vertex Kernel as sudo**:
    ```bash
    sudo go run ./cmd/kernel --user $(whoami)
    ```

    or if you want to use a custom host/port:
    ```bash
    sudo go run ./cmd/kernel \
        --user $(whoami) \
        --host localhost \
        --port 6130
    ```

4. Open the UI in your browser: [http://localhost:6130](http://localhost:6130)

---

## Install the Client

1. Install Node.js using the [official documentation](https://nodejs.org/en/download/).

2. Download the project
    ```bash
    git clone https://github.com/vertex-center/client.git vertex-client
    cd vertex-client
    ```

3. Install dependencies
    ```bash
    npm install
    ```

4. Run the client
    ```bash
    npm run dev
    ```
