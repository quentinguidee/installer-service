# Security

## SSH keys

This menu allows you to manage your SSH keys, without having to access your server manually.

All SSH keys added here will be added to the `~/.ssh/authorized_keys` file on your server.

### List SSH keys

All SSH keys added to your server are listed on this security page.

### Add a new SSH key

To add a new SSH key:

1. Click on the "Create an SSH key" button.
2. Generate an SSH key if you don't have one yet.
3. Copy the public key and paste it in the "Public key" field.
